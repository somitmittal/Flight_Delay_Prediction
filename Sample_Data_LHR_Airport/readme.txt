Columns in the data: 

departure_date - Date of departure
airline_code - Airline Code (for example: British Airways as BA, Air France as AF etc). 
flight_number - Flight Number (this is string. Can contain non-integer characters)
source_airport - Airport code from where flight departs
destination_airport - Airport code where flight lands
departure_scheduled_time - scheduled time of departure (epoch)
departure_actual_time - actual time of departure (epoch)
arrival_scheduled_time - scheduled time of arrival (epoch)
arrival_actual_time - actual time of arrival (epoch)
